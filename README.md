# Two-Part Abbreviation Typographer

Corrects whitespaces in two-part abbreviations, e.g. `e.g.` or `и т. д.`.

Input|→|Output
:---|---|:---
`e. g.`|→|`e.g.`
`a. m.`|→|`a.m.`
`и т.д.`|→|`и т. д.`
`и т.п.`|→|`и т. п.`
`ст.л.`|→|`ст. л.`

See the live [demo].
